# src/hooks/ ‚Äî Custom React hooks

| Hook             | Description                                   |
|------------------|-----------------------------------------------|
| `useMediaQuery`  | Reactive CSS media query matcher              |
| `use-mobile.ts`  | `useIsMobile()` ‚Äî shorthand for 768px query   |
| `useFocusTrap`   | Trap keyboard focus inside a container (modals)|
| `useLockScroll`   | Lock body scroll (for overlays, menus)        |
| `use-toast.ts`   | Toast notification state manager              |

All hooks are client-side (`"use client"`).
